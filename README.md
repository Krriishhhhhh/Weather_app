# Weather App

Welcome to the Weather App! This application provides real-time weather information for User's locations.

## Tech Stack

- **Next.js 13:** A React framework providing server-side rendering and other powerful features.
- **TypeScript:** Adds static typing to JavaScript, enhancing code quality and developer experience.
- **Express:** A flexible Node.js web application framework used for building the backend API.
- **Tailwind CSS:** A utility-first CSS framework for creating custom designs.
- **Prisma:** A modern database toolkit for accessing databases through a type-safe and fluent API.
- **PostgreSQL:** A powerful, open-source relational database system used to store application data.
